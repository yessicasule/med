# Medical Care Management System

The Medical Care Management System is a web-based application designed to streamline healthcare operations through dedicated portals for patients, doctors, receptionists, and administrators. It centralizes appointment handling, medical records, billing, and user management in a single platform.

## Features

### Patient Portal
- Search for doctors
- Book and manage appointments
- Access medical records and prescriptions
- View invoices and payment status

### Doctor Portal
- View daily appointment schedule
- Access patient history
- Upload prescriptions and medical records

### Receptionist Portal
- Manage appointments for all doctors
- Generate bills and record payments
- Access patient information and history

## Technology Stack
- React, TypeScript, Vite
- React Router, React Query
- Tailwind CSS, shadcn/ui
- LocalStorage-based data persistence

## System Architecture
- Modular databases for users, doctors, appointments, billing, medical records, and notifications
- JWT-based authentication and role-based access control
- Service-layer architecture for business logic
- Real-time data synchronization using React Query

## Summary
This system improves operational efficiency for healthcare facilities by providing a unified, secure, and role-based platform for managing appointments, records, billing, and administrative workflows.
